# Pomodoro Timer

공부나 작업할 때 집중력을 높여주는 **Pomodoro 타이머 스크립트**입니다.  
집중 시간과 휴식 시간을 설정하고, 원하는 횟수만큼 반복할 수 있습니다.

---

## 기능
- **집중 시간 설정**: 기본 25분
- **휴식 시간 설정**: 기본 5분
- **반복 횟수 설정**: 기본 4회
- 간단한 CLI 인터페이스 제공

---

## 설치 및 실행 방법

1. 레포지토리 클론
   ```bash
   git clone https://github.com/username/PomodoroTimer.git
   cd PomodoroTimer
2. 스크립트 실행 Bash셸 -> python pomodoro.py
3. 입력값
- 집중시간 집중 시간(분)
- 휴식 시간(분)
- 반복 횟수

사용예시:
- Pomodoro 타이머를 시작합니다.
- 집중 시간(분): 25
- 휴식 시간(분): 5
- 반복 횟수: 4

결과:
- 🍅 1번째 집중 시작! 25분 동안 집중하세요.
- ☕ 휴식 시간! 5분 동안 쉬세요.
...
- 🎉 모든 Pomodoro 세션이 끝났습니다! 수고하셨습니다.

요구사항
- Python 3.x
- 추가 라이브러리 없음 (표준 라이브러리만 사용)
